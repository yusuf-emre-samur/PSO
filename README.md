# PSO

## Build:
```cd build```

```cmake -S ../ -B .```

```cmake .```

```make```

## Run:
```./build/main $population_size $ iterations $v_max $x_spawn $function_no```

## Functions:
1. Rosenbrock Function
2. Drop Wave Function
3. Levy N13 Function
4. Ackley Function

## Requires:
Eigen Library in path: ```/usr/include/eigen3```
